## Description
Welcome to my portfolio project! This portfolio showcases a 
gallery featuring six distinct projects that I've developed.
Each project demonstrates various skills and functionalities.
The projects included are:

1. Snake Game
2. Memory Game
3. Tic Tac Toe Game
4. Photo Gallery
5. Calculator
6. To-Do List

This portfolio provides the option to explore these projects either by downloading them as ZIP files or by using them directly
within your browser on separate pages. 
Each project highlights different aspects of web development and programming.
The primary purpose of this portfolio is to share my work and give you the opportunity
to interact with these projects. Enjoy exploring!

## Installation

1. Clone this repository: `git clone https://github.com/KarinGanon/Project.git`
2. Navigate to the project directory: 'pnpm dev' to run the local host of vite. 

## Usage

1. Open the `index.html` file in your web browser.
2. Browse through the sections to learn more about me, view my projects, and find my contact information.
